# よしもとメンバー NSC期別早見表

渋谷よしもと漫才劇場（東京）とよしもと漫才劇場（大阪）の所属メンバーをNSC期別に一覧表示する非公式早見表サイトです。

## サイト

https://p-dev12.github.io/yoshimoto-nsc/

## 機能

- 東京・大阪メンバーの切り替え
- 極・翔メンバーのフィルタリング
- コンビ名検索
- NSC期・年目の自動表示
- 毎日午前3時に自動更新

## 構成

本リポジトリはサイトの公開用です。
自動更新スクリプト（`update.py` / `build.py`）はPrivateリポジトリで管理しています。

## 情報出典

- [よしもと漫才劇場（大阪）](https://manzaigekijyo.yoshimoto.co.jp/profile/)
